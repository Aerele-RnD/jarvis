# Documentation has moved

Jarvis documentation is now maintained in one place — the **`jarvis_admin`**
repo, under **`jarvis_admin/docs/`**:

- Customer-app docs (this app): `jarvis_admin/docs/customer-app/`
  (getting-started, architecture, configuration, tools-api, local-dev,
  development, decisions).
- Operator/platform docs: `jarvis_admin/docs/` (start at `production-deploy.md`).

> Note: `jarvis_admin` is an internal Aerele repo. This consolidation keeps all
> docs together for the team; if customer-facing docs need to ship publicly
> later, publish the `customer-app/` set from there.
